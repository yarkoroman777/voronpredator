# Triangular Arbitrage Bot for Binance

## Описание
Бот для треугольного арбитража на Binance (USDT → BTC → ETH → USDT).  
Использует WebSocket для получения цен, проверку глубины стакана, лимитные ордера и уведомления в Telegram.
